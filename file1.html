Sets

  • A set is a well defined collection of distinct objects.(elements)
  • Well-defined: This means there is no confusion about what belongs. (e.g., "Good books" is not a set,
    but "Books written by Shakespeare" is a set).
  • Elements: The individual objects in a set,
    usually denoted by lowercase letters (a, b, c), while the set itself is named with capital letters (A, B, C).

Description of Sets

  • Roster/Tabular Form: Listing every element inside curly brackets.Example: A = {2, 4, 6, 8}
  • Description Method: Writing a short phrase to describe the members.Example: A ={Even numbers less than 10}
  • Set-Builder Form: Using a rule or variable to define elements.Example: A = \{x | x is an even natural number, x < 10}
  • Cardinal Number n(A): The total number of unique elements in Set A.

  • Empty (Null) Set (âˆ…): A set with no elements.
  • Singleton Set: A set containing only one element.
  • Finite Set: A set with a countable number of elements
  • Infinite Set: A set with an uncountable number of elements.
  • Equal Sets: Two sets that contain exactly the same elements.
  • Equivalent Sets: Sets that have the same number of elements, even if the elements are different.
