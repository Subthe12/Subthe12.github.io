Integers

About Integers

  • Integers are whole numbers that can be positive, negative, or zero.
  • They do not include fractions or decimals.
  • Examples of integers: -3, -2, -1, 0, 1, 2, 3
  • The numbers 1, 2, 3… are called positive integers.
  • The numbers -1,-2,-3… are called negative integers.
  • Zero is neither positive nor negative.
  • Integers are represented on a number line, with negative integers to the left of zero and positive integers to the right.

A number line

Integers are used in various real-life situations, such as measuring temperature (above and below zero), representing financial transactions (gains and losses), and indicating elevations (above and below sea level).

Addition and Subtraction of Integers

  • To add two positive integer: add then as natural numbers. (1+1=2)
  • To add two negative numbers: ignore the signs, add them as natural numbers,
    and then add the negative sign. (-2+-6=-8)
  • To add a positive and a negative number: ignore their signs,
    subtract the smaller number from the larger number,
    and then give the sign of the integer which has the greater absolute value. {4+-8=-(8-4) =-4}
  • The subtraction of 2 integers in same as adding the additive inverse of the integer that is being subtracted to the other integer.
    If a and b are two integers then,a – b = a + additive inverse of b = a + (-b) and a – (-b) = a + additive inverse of (-b) = a + b

Sets

  • A set is a well defined collection of distinct objects.(elements)
  • Well-defined: This means there is no confusion about what belongs. (e.g., "Good books" is not a set,
    but "Books written by Shakespeare" is a set).
  • Elements: The individual objects in a set,
    usually denoted by lowercase letters (a, b, c), while the set itself is named with capital letters (A, B, C).

Description of Sets

  • Roster/Tabular Form: Listing every element inside curly brackets.Example: A = {2, 4, 6, 8}
  • Description Method: Writing a short phrase to describe the members.Example: A ={Even numbers less than 10}
  • Set-Builder Form: Using a rule or variable to define elements.Example: A = \{x | x is an even natural number, x < 10}
  • Cardinal Number n(A): The total number of unique elements in Set A.

Types of Sets

  • Empty (Null) Set (∅): A set with no elements.
  • Singleton Set: A set containing only one element.
  • Finite Set: A set with a countable number of elements
  • Infinite Set: A set with an uncountable number of elements.
  • Equal Sets: Two sets that contain exactly the same elements.
  • Equivalent Sets: Sets that have the same number of elements, even if the elements are different.
